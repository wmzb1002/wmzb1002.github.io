智慧中小学教材下载器

Latest v0.0.2 | 专为学习设计 · 非商业用途

本程序旨在帮助学生教师学习使用，仅用于教育研究目的。我们完全支持国家教育政策，尊重版权规范。

⚠️ 重要声明：请勿将本工具用于任何商业用途！本程序仅供学习参考，不得传播非法内容或侵犯他人权益。

📌 主要功能

  • 支持从“智慧中小学”平台解析并下载教材PDF
  • 自动提取文件名，简化操作流程
  • 流式下载，节省内存
  • 增强异常处理，提升稳定性
  • 兼容最新认证 headers 参数

🚀 使用方法

  1. 访问智慧中小学官网，打开所需教材页面
  2. 按下 F12 或右键 → “检查” 打开开发者工具
  3. 切换到 网络(Network) 选项卡
  4. 筛选类型为 文档(Document)，并在搜索框输入 pdf
  5. 刷新页面，找到对应的PDF请求条目
  6. 右键复制其 链接URL
  7. 将完整链接粘贴至程序输入框中即可开始下载 ✨
